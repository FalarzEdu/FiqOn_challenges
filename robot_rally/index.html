ROBOT RALLY

Instruções

O Rally do robôs é um pequeno jogo onde você deve criar uma arena com obstáculos e recompensas, fornecendo também os comandos para que o robô colete todos os prêmios. Como saída, o programa retornará o número de recompensas coletadas.

Você deve fornecer o número de colunas e linhas da arena, bem como a posição inicial,orientação do robô, número e sequência de comandos. Com todos os valores inseridos de forma válida, basta clicar no botão "Start".

Certifique-se de que a arena está com o número de linhas e colunas correto.

A arena é composta por 3 elementos:

  • Bloco vazio: . (ponto)
  • Obstáculo: # (hashtag)
  • Recompensa: * (asterisco)

O robô possui 4 representações que também indicam sua orientação (norte, sul, leste e oeste).

  • Norte: N
  • Sul: S
  • Leste: L
  • Oeste: O

Existem 3 tipos de comandos válidos:

  • Para frente: F
  • 90º para a esquerda: E
  • 90º para a direita: D
